Emdepub is an Eclipse plugin for displaying and editing Markdown (.md) files, and creating ePub (.epub) books.

ADDED SUPPORT FOR TOML

The plugin now has support for TOML, with syntax, verification and proposals

This plugin is a new version of the now deprecated (no longer maintained) plugin Markdown Semantic EP (GitHub). The difference between the two comes from the fact that the visual part is now done in the Eclipse "modern" way, with tm4e, JavaScript render, and MS VSC. The code manipulation part has remain in Flexmark and for a detailed description of each functionality, please go to Markdown progress

The main goals are speed and quality for both the syntax highlight and the resulted HTML. These qualities still doesn't seem to be entirely fulfilled by the existing Eclipse plugins.

For using the ePub creator, plase see the example inside the GitHub sources.

Install

Install as a normal Eclipse plugin:

Help -> Install new software...

And then add the site https://emdepub.org/update
